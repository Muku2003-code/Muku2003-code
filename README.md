# 🌍 Global Cybersecurity Threats Analysis

An exploratory data analysis project focusing on the identification and visualization of cybersecurity threats globally. This notebook-driven project utilizes real-world data to uncover key insights on types of attacks, top affected countries, and annual trends in global cyber threats.

---

## 📌 Objective

This project aims to:
- Analyze global cybersecurity incidents.
- Identify countries most affected by different types of cyberattacks.
- Examine yearly trends in cybersecurity threats.
- Visualize threat categories and patterns using interactive plots.

---

## 📁 Dataset Overview

The dataset used in this project was sourced from Kaggle and provides data on cybersecurity threats from multiple countries and years.  
It includes columns like:
- `Country`
- `Year`
- `Threat Type`
- `Count`

📂 **Dataset Source:**  
[🔗 Global Cybersecurity Threats Dataset – Kaggle](https://www.kaggle.com/code/devraai/global-cybersecurity-threats-analysis/input)

---

## 🧰 Tools & Libraries Used

- **Python 3**
- **Pandas** – data manipulation
- **Matplotlib & Seaborn** – data visualization
- **Plotly** – interactive plots
- **Jupyter Notebook** – for exploratory analysis

---

## 🔍 Key Analyses Performed

- **Top Countries by Cybersecurity Threats**  
  Visualized the most affected countries using bar charts.

- **Yearly Threat Trends**  
  Analyzed changes in threat frequency over time.

- **Threat Category Distribution**  
  Pie chart showing proportions of threat types.

- **Correlation & Heatmaps**  
  Explored relationships among dataset features.

---

## 📊 Visual Highlights

Key visualizations include:
- Bar Plot: Top 10 affected countries
- Pie Chart: Distribution of threat categories
- Line Graph: Year-wise global trends
- Heatmap: Correlation matrix of numerical features
